## How These Prompts Work Together

Each model has a specialized role in the orchestration mesh. The prompts are designed to be chained:

1. **Gemini** builds the macro context (Macro-Context Synthesis Engine) and discovers cross-domain signals (Cross-Domain Signal Discovery).
2. **DeepSeek** takes Gemini's true probability estimates and runs them through the Edge Identification Engine, then sizes positions with the Kelly Criterion Optimizer.
3. **Venice AI** challenges the consensus — its Contrarian Edge Finder and Strake Pattern Detector look for non-linear edges that the linear models miss.
4. **Gemini's** Shop-Scales Portfolio Balancer maintains risk equilibrium across all identified positions.

```
[Gemini: Macro Context] → [DeepSeek: Edge + Kelly] → [Venice: Contrarian + Strake] → [Gemini: Shop-Scales Balance]
```

## Usage

1. **Copy** any prompt from the model-specific files above.
2. **Paste** into the corresponding AI model's system prompt or chat interface.
3. **Provide** the input data specified in each prompt's "Given" section.
4. **Chain** outputs between models as described in each file's Cross-References section.

For the interactive web version with copy-to-clipboard buttons and tab navigation, visit the [Emberscot Prompt Library page](https://emberscot.com/prompt-library).
